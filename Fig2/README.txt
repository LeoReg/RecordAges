Codes for generating the data used in Fig 2 (the record ages):
-Fig2a.nb: the mathematica code for generating trajectories of fbm.
-Fig2b.py: the python code for generating trajectories of quenched fbm.
-Fig2c.py: the python code for generating trajectories of elephant random walks.
-Fig2def.py: the python code for generating the SATW, the SESRW and TSAW by varing the value of $\kappa$ (0, 0.5 and 1)
-Fig2g.py: the python code for generating trajectories of the subdiffusive average Lévy Lorentz walk.
 -Fig2h.py: the python code for generating trajectories of the superdiffusive average Lévy Lorentz walk.
